Become a productive AngularJS developer
  • Be able to write robust and scalable enterprise apps
  • Understand the deep rooted concepts of AngularJS (directives, routing, forms & validation, architecture...)
  • Use the right syntax and design to build performant Single Page Applications
For whom?

The 3 days Angular MasterClass is intended for developers. All participants should have at least a good knowledge of or more programming languages (Java, PHP, C#, C++, Python, Scala, Go ...) and a basic understanding of JavaScript.

What you will learn

Build our own Single Page Application

  • JS fundamental for AngularJS
  • Core concepts in Angular (controller, scope, Service & Factory, filters, directives)
  • Putting things together by building a simple application.
  • Built-in and custom services
  • Build tools with Angular (Gulp, Browserify, Webpack, Surge)

Everything is a component

  • Different types of directives
  • Advanced directives
  • Building your custom directives
  • Understanding the different types of scope and when to use them
  • Advanced routing
  • Testing your Angular components
  • Best practices for building components based applications
  • Refactor your application to use a components based architecture

Forms, Unit testing, E2E testing, Performances, Angular 2

  • Forms and validations in Angular
  • Advanced unit testing strategy
  • End to End testing with Protractor
  • Performances in Angular
  • Internalisation of our application
